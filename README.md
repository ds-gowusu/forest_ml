Project Introduction

This project, ForestYieldML, focuses on developing and comparing machine learning regression models for 
predicting plantation growth using Permanent Sample Plot (PSP) data. Accurate estimation of forest stand
metrics - such as tree volume, height, and basal area - is essential for effective forest management, yield
planning, and decision-making in plantation forestry.

The study adopts a comparative modeling approach, implementing four regression techniques: Linear Regression
as a baseline model, Random Forest and XGBoost as ensemble learning methods, and Support Vector Regression as
a kernel-based approach. These models are evaluated to understand their predictive performance and 
suitability for forestry applications.

By leveraging PSP data, which captures long-term growth measurements for forest stands, this project aims to 
demonstrate how data-driven methods can improve the accuracy of reliability of forest inventory predictions. 
The results provide insights into the strengths and limitations of different regression techniques in modeling
complex biological growth patterns.
